# 视频解构分析器 - Video Deconstruction Analyzer

上传视频，AI自动分析工具/效果/素材/提示词，生成专业分镜脚本。

## 功能
- 工具识别 AE/Premiere/DaVinci/CapCut
- 效果拆分 转场/粒子/光效/调色
- 素材分析 实拍/AI生成判断
- 复刻提示词 MJ/Runway/Pika/Kling
- 逐帧分析 11维度结构化
- 分镜脚本 Shot-by-Shot
- 多模型 12+ API提供商
- 多格式导出 Seedance/Runway/Markdown/JSON
